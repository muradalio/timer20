# Timer20

Нативный macOS menu bar таймер для цикла 20 минут работы и 20 секунд отдыха.

Автор: Mur.

Интерфейс автоматически переключается на английский, если основной язык системы английский.

## Запуск

```sh
make run
```

После запуска приложение появится в строке меню рядом с часами. Иконки в Dock нет.

## Сборка `.app`

```sh
make app
```

Готовое приложение будет здесь:

```text
build/Timer20.app
```

## Установка в Applications

```sh
make install
```

После установки приложение будет здесь:

```text
/Applications/Timer20.app
```

Запуск установленной версии:

```sh
make run-installed
```

## Поведение

- 20 минут работы.
- Уведомление: пора отдохнуть.
- 20 секунд отдыха.
- Уведомление: можно продолжать.
- Цикл повторяется автоматически.
- Длительности можно изменить в настройках.
- Автозапуск при входе в macOS включается в настройках.

В меню доступны пауза, ручной старт отдыха, сброс, настройки, информация о версии и выход.

Приложение использует template-иконку в строке меню и подстраивается под светлую/тёмную тему macOS.

## Настройки

Откройте меню `Timer20` в строке меню и выберите `Настройки...`.

Доступные параметры:

- длительность работы в минутах;
- длительность отдыха в секундах;
- запуск при входе в macOS.

После сохранения таймер начинает новый рабочий цикл с актуальными значениями.
